1. Declaração de Arquitectura e Fluxo de Dados

⚠️ DECLARAÇÃO TÉCNICA OBRIGATÓRIA (para efeitos de admissibilidade pericial):

Este sistema opera com uma interface local (ficheiro HTML), mas possui capacidade de processar inputs textuais através de uma ponte API externa (Proxy via api.unifed.com → api.anthropic.com/v1/messages) para estruturação de dados por um Modelo de Linguagem (LLM — Anthropic Claude).

O sistema NÃO é uma sandbox isolada. Dados introduzidos pelo utilizador e processados através das funções de análise por IA são transmitidos para servidores externos. O perito ou mandatário deve considerar este facto ao avaliar a admissibilidade de dados processados através dessas funções em sede judicial.

A bateria de 50 questões de contraditório (componente principal desta ferramenta) não realiza comunicações externas — opera exclusivamente com a base de dados local embutida neste ficheiro HTML.
